Edusphere – College Portal Website
Overview

Edusphere is a responsive college portal website developed for students and teachers with a focus on structured layouts, easy navigation, and user-friendly design. The project was created to provide a clean and accessible web interface for academic-related activities.

Features
Responsive web interface
Structured and clean page layouts
Student and teacher portal pages
User-friendly navigation
Improved readability and accessibility
Interactive UI components
Technologies Used
HTML
CSS
Basic JavaScript
Project Goals
Build a responsive frontend interface
Improve usability and navigation experience
Practice frontend development concepts using HTML, CSS, and JavaScript
Create a clean and visually organized web layout
